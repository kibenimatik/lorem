Lorem

Ever get tired of opening up lipsum.com, filling out a form and waiting for it to give you your filler text?

This is the sample Layerment app written on Ruby. It provides API for generating lipsum text.

Usage

/lorem
/lorem?number&type
/lorem?n&t

query parameters

number - Defaults to 1 if no parameters given, shortcut is "n"

type - Defaults to paragraphs, available types: "paragraphs", "words", "chars", "characters". Shortcut is "t"

Examples

/lorem - generate 1 paragraph

/lorem?number=5 - generate 5 paragraphs

/lorem?number=5&type=words - generate 10 words

/lorem?n=100&t=chars - generate 100 characters
